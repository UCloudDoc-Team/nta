# 新功能发布记录

本文为网络流量分析（Network Traffic Analytics，简称NTA）产品的主要功能发布记录。

| **发布时间** | **功能名称** | **变更类型** | **功能描述**                                                 |
| ------------ | ------------ | ------------ | ------------------------------------------------------------ |
| 2026年2月    | 跨域流量分析 | 新增         | 面向云联网场景的跨域流量分析工具，目前仅支持华东（上海2）    |
| 2026年6月    | 跨域流量分析 | 优化         | 面向云联网场景的跨域流量分析工具，新增支持华北（北京2）、华南（广州）、香港地域 |
