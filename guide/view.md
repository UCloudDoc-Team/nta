# 查看流量分析

1. 登录[UCloud控制台](https://passport.ucloud.cn/#login)；
2. 选择某个流量分析器实例，单击操作列选择详情
3. 可查看该分析器的概览信息、流量分析数据及操作日志等核心内容。

## **概览**

点击概览，查看流量分析器基本信息，包括资源名称、状态、统计策略、资源信息等。

![概览](images/概览.png)

## **流量分析**

点击流量分析，查看该流量分析器基于预设的统计配置策略所生成的**流量分析数据**。

### 一元组

> 一元组展示带宽包实例维度的Top流量统计

1. 在筛选区域选择地域、带宽包、时间范围（支持自定义时间范围，最大跨度不超过1天）

2. 支持查看统计/趋势图表

   - 出向跨域流量趋势：展示筛选范围内所有云联网带宽包在目标时间范围内的出向跨域流量波动趋势。
   - 出向跨域流量TOP 10排行：展示筛选范围内出向流量排名前10的云联网带宽包数据。

3. 支持查看流量列表

   - 点击操作列查看趋势：支持查看某个本端IP地址在目标时间段内的出向流量波动趋势、出向包数波动趋势。
   - 点击操作列下钻二元组（需开启二元组分析功能）：支持查看该本端IP地址在目标时间段内与各对端IP的二元组流量统计数据。

   ![一元组](images/一元组.png)

#### 二元组

> 二元组展示IP维度的Top流量统计

1. 在筛选区域选择地域、带宽包、IP信息、时间范围（支持自定义时间范围，最大跨度不超过30天）

2. 支持查看统计/趋势图表

   - IP 出向带宽 TOP 10：展示筛选范围内，本端IP地址-对端IP地址 TOP10 的跨域流量趋势
   - 访问对端流量 TOP10：展示筛选范围内，本端IP地址-对端IP地址 TOP10 的跨域流量排行

3. 支持查看流量列表

   - 点击操作列查看趋势：支持查看某个本端IP地址-对端IP地址在目标时间段内的出向流量波动趋势、出向包数波动趋势。

   ![二元组](images/二元组.png)

## **操作日志**

点击操作日志，可查看流量分析器的生命周期操作日志，涵盖**创建、更新、删除等**关键操作行为，具体信息可查看[操作日志 ULog_文档中心](https://docs.ucloud.cn/manage/activity)
